# AI Study Assistant (Generative AI Project)

## 📌 About Project
AI Study Assistant is a Generative AI based web application that helps students by answering questions, summarizing notes, and generating quiz questions instantly.

## 🚀 Features
- Ask AI doubts (Chat feature)
- Summarize long notes into short points
- Generate quiz questions with answers
- Clear chat option

## 🛠️ Technologies Used
- HTML
- CSS
- JavaScript
- Google Gemini API (Generative AI)
